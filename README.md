# 看不见的废墟：静态网页草稿

这是第十一届中国数据新闻大赛作品的网页草稿。当前版本已拆分为首页与五个章节页，可直接部署到 GitHub Pages。

## 页面结构

- `index.html`：首页与全球电子废弃物增长计时。
- `global.html`：全球规模、材料账本、正式记录回收率。
- `china.html`：跨国统一口径下的中国总量压力，以及国家统计局口径下的中国电子电器产品产量和数字生活规模。
- `missing.html`：中国电子废弃物产生量、正式收集量与统计缺口。
- `unequal.html`：电子废弃物链条中的健康风险与风险转移。
- `afterlife.html`：旧设备的四种去向与互动路径。

## 已用主要数据源

- 联合国训练研究所（United Nations Institute for Training and Research）与国际电信联盟（International Telecommunication Union）：《Global E-waste Monitor 2024》
- 世界卫生组织（World Health Organization）：《Children and digital dumpsites: e-waste exposure and child health》
- 国家统计局：《2023年国民经济和社会发展统计公报》
- 国家统计局：《2024年国民经济和社会发展统计公报》
